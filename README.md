# 은퇴현금 서비스 기획서

## 1. 서비스 개요

`vibe-retirement`는 은퇴 예정자가 은퇴 후 월별 수입과 지출을 확인하고, 건강보험료·연금·자산 고갈 가능성을 함께 검토할 수 있도록 돕는 은퇴현금 설계 서비스입니다.

초기 방향은 종합 은퇴 플랫폼이지만, MVP 1차에서는 사용자가 가장 먼저 확인하고 싶어 하는 질문에 집중합니다.

> 은퇴 후 매달 생활비가 충분한가?
> 내 자산은 몇 세까지 버틸 수 있는가?
> 건강보험료 같은 고정지출을 감안했는가?

## 2. 서비스 대상

- 40세 ~ 60세 은퇴 예정자
- 은퇴 후 재정 계획에 대한 막연한 불안감이 있는 사람
- 퇴직 후 월별 현금흐름을 확인하고 싶은 사람
- 건강보험료, 연금, 생활비, 의료비를 함께 보고 싶은 사람
- 은퇴 자산이 언제 부족해질 수 있는지 미리 알고 싶은 사람

## 3. 핵심 가치

- **현금흐름 중심:** 은퇴 후 월별 수입/지출/순현금흐름을 확인
- **불안 해소:** 자산 고갈 가능성과 부족 시점을 미리 파악
- **쉬운 이해:** 복잡한 금융·세무 정보를 사용자가 이해하기 쉬운 형태로 제공
- **실행 중심:** 생활비 조정, 연금 수령, 건강보험료 확인 등 다음 행동 안내
- **확장 가능성:** ISA, 연금, ETF, 전문가 상담, 체크리스트 기능으로 단계적 확장

## 4. MVP 1차 범위

심층 분석 결과, 초기 MVP는 아래 기능에 집중합니다.

- **F-01** 개인별 은퇴 목표 및 현재 재정 상태 입력
- **F-02** 건강보험료 예상 시뮬레이션
- **F-09** 은퇴 후 현금흐름 시뮬레이션
- **F-10** 은퇴 자산 고갈 시점 예측
- **F-11** 은퇴 계획 진행 상황 대시보드

## 5. 2차 확장 기능

- **F-03** ISA 계좌 개설 및 운용 가이드
- **F-04** ISA 계좌를 통한 종합소득세 절세 효과 분석
- **F-05** 퇴직연금(DC/DB/IRP) 현황 분석 및 최적화 전략
- **F-06** 개인연금(연금저축/연금보험) 포트폴리오 진단
- **F-07** 은퇴 자산 운용을 위한 ETF 포트폴리오 구성
- **F-08** 투자 성향 분석

## 6. 3차 고도화 기능

- **F-12** 전문가 상담 연결
- **F-13** 은퇴 관련 최신 세법 및 금융 상품 정보 제공
- **F-14** 은퇴 준비 체크리스트 및 목표 달성 알림

## 8. 개발 우선순위

1. 공통 데이터 모델 정의
2. F-01 입력 화면 구현
3. F-09 월별 현금흐름 계산 구현
4. F-10 자산 고갈 시점 계산 구현
5. F-02 건강보험료 추정 구현
6. F-11 대시보드 구현
7. F-03~F-14 단계적 확장

## 9. 중요 고지

본 서비스의 세금, 건강보험료, 연금, 투자 관련 결과는 사용자가 입력한 정보와 계산 가정에 기반한 추정 결과입니다. 실제 금액과 결과는 개인 상황, 제도 변경, 금융기관 기준에 따라 달라질 수 있습니다.

최종 의사결정 전에는 국민건강보험공단, 국세청, 금융기관 또는 전문가 확인이 필요합니다.

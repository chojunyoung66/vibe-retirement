# F-04: ISA 계좌를 통한 종합소득세 절세 효과 분석

## 1. 기능 개요

본 기능은 ISA 계좌 운용을 통해 발생할 수 있는 종합소득세 절세 효과를 구체적으로 분석하고 시뮬레이션하여 사용자에게 제시합니다. 특히 은퇴 후 금융 소득이 발생할 경우 종합소득세 합산 과세로 인한 세금 부담을 ISA 계좌가 어떻게 경감시킬 수 있는지 보여줍니다.

## 2. 상세 설명

사용자가 F-01 기능을 통해 입력한 재정 정보와 F-03에서 제시된 ISA 계좌 운용 전략을 바탕으로, ISA 계좌를 활용했을 때와 활용하지 않았을 때의 종합소득세 부담을 비교 분석합니다. 주요 분석 내용은 다음과 같습니다:

*   **금융 소득 종합과세 이해:** 은퇴 후 발생할 수 있는 이자 소득, 배당 소득 등 금융 소득이 연간 2천만원을 초과할 경우 종합소득세에 합산되어 과세되는 원리를 설명합니다.
*   **ISA 계좌 비과세/분리과세 효과:** ISA 계좌 내에서 발생한 수익에 대해 비과세 한도(서민형/농어민형 400만원, 일반형 200만원)를 초과하는 금액에 대해서는 9.9%의 낮은 세율로 분리과세되는 효과를 시뮬레이션합니다.
*   **종합소득세 절감액 예측:** ISA 계좌를 통해 금융 소득이 비과세되거나 분리과세됨으로써 절감되는 종합소득세액을 구체적인 수치로 제시합니다.
*   **건강보험료 연계 분석:** 금융 소득이 종합소득세에 합산되지 않음으로써 건강보험료 산정 시 소득 반영이 줄어들어 건강보험료까지 절감될 수 있는 연계 효과를 분석합니다.
*   **최적의 ISA 운용 전략 제안:** 사용자의 예상 금융 소득 규모와 은퇴 시점을 고려하여 ISA 계좌의 납입 한도, 운용 기간 등을 최적화하는 전략을 제안합니다.

이 기능은 ISA 계좌를 통한 종합소득세 절세 효과를 명확하게 보여줌으로써, 사용자가 은퇴 후 재정 계획을 수립하는 데 있어 ISA 계좌의 중요성을 인지하고 적극적으로 활용하도록 유도합니다.

## 3. 기대 효과

*   ISA 계좌의 세금 절감 효과를 구체적인 수치로 제시하여 사용자의 이해도를 높입니다.
*   은퇴 후 발생할 수 있는 종합소득세 부담을 효과적으로 관리할 수 있는 방안을 제공합니다.
*   건강보험료 절감 효과까지 연계하여 ISA 계좌의 활용 가치를 극대화합니다.

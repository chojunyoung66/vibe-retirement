- 디테일한 분석과 시뮬레이션

## 3. vibe-retirement에 반영할 관점

### 3.1. 한 페이지 요약 결과

사용자는 복잡한 은퇴 정보를 길게 보고 싶어 하기보다, 핵심 결과를 한 페이지에서 확인하고 싶어 합니다.

F-11 대시보드에는 다음 정보를 한 화면에 요약합니다.

- 월 예상 수입
- 월 예상 지출
- 월 순현금흐름
- 총 예상 연금 수입
- 3층 연금 요약
- 건강보험료 추정치
- 자산 고갈 예상 시점
- 부족 현금 발생 여부
- 다음에 조정할 항목

### 3.2. 자산 시나리오 비교

후기에서 “다양한 시나리오”와 “미래 예측”에 대한 만족이 확인됩니다.

F-09와 F-10에서는 최소 3개 시나리오를 제공합니다.

- 기본 시나리오
- 생활비 증가 시나리오
- 수익률 하락 시나리오

추가 확장 시나리오:

- 의료비 증가
- 국민연금 수령 시점 변경
- 퇴직연금 인출 방식 변경
- 배우자 국민연금 반영

### 3.3. 인출 전략의 중요성

은퇴 후에는 자산을 얼마나 모았는지만큼, 언제 어떻게 인출할지가 중요합니다.

F-09 현금흐름 시뮬레이션에서는 다음을 구분합니다.

- 정기 수입: 국민연금, 퇴직연금, 개인연금
- 인출 재원: 금융자산, ISA, IRP, 예금
- 고정 지출: 생활비, 건강보험료, 보험료, 재산세
- 변동 지출: 의료비, 경조사비, 여가비

### 3.4. 절세계좌와 세금 관점

후기에서 절세계좌와 세금 측면의 도움이 언급됩니다.

2차 기능에서 다음을 반영합니다.

- F-03 ISA 계좌 개설 및 운용 가이드
- F-04 ISA 절세 효과 분석
- F-05 퇴직연금 세제와 수령 방식
- F-06 개인연금 세액공제와 수령 전략

단, 초기 MVP에서는 복잡한 세금 계산보다 안내와 추정 중심으로 제한합니다.

### 3.5. ETF 투자 주의점

ETF는 은퇴 자산 운용 수단이 될 수 있지만, 원금 손실과 변동성 위험이 있습니다.

F-07과 F-08에서는 다음 원칙을 지킵니다.

- 개별 ETF 매수 권유보다 자산군 예시 중심
- 투자 성향 진단 후 포트폴리오 예시 제공
- 안정형에 고위험 상품을 배치하지 않음
- 세금, 수수료, 환율, 추적오차, 변동성 위험 고지

## 4. 기능별 반영 사항

| 기능 | 반영 내용 |
|---|---|
| F-01 | 국민연금, 퇴직연금, 개인연금, 금융자산, 지출을 한 번에 입력 |
| F-02 | 건강보험료를 은퇴 후 고정지출로 명확히 표시 |
| F-03 | ISA 등 절세계좌를 은퇴 현금흐름 관점에서 안내 |
| F-04 | 세금 절감 효과는 추정치로 표시 |
| F-05 | 퇴직연금은 수령 방식과 인출 전략을 함께 검토 |
| F-06 | 개인연금은 3층 연금 보완 수단으로 정리 |
| F-07 | ETF는 상품 추천보다 자산군 배분 예시로 제한 |
| F-08 | 투자 성향 진단 후 F-07로 연결 |
| F-09 | 월별 수입, 지출, 인출 재원을 구분해 시뮬레이션 |
| F-10 | 자산 고갈 시점을 시나리오별로 비교 |
| F-11 | 한 페이지 요약 대시보드 제공 |
| F-12 | 복잡한 의사결정은 전문가 상담으로 연결 |
| F-13 | 세법과 금융상품 정보는 출처와 기준일 표시 |
| F-14 | 은퇴 준비 실행 항목을 체크리스트로 관리 |

## 5. MVP 반영 우선순위

1. F-11 한 페이지 요약 대시보드
2. F-09 월별 현금흐름 시뮬레이션
3. F-10 자산 고갈 시나리오 비교
4. F-01 3층 연금 및 지출 입력
5. F-02 건강보험료 고정지출 반영

## 6. 완료 기준

- F-11 결과 화면에서 핵심 결과를 한 페이지로 볼 수 있다.
- F-09에서 수입, 지출, 인출 재원을 구분한다.
- F-10에서 최소 3개 시나리오를 비교한다.
- 3층 연금 중 국민연금을 반드시 포함한다.
- ETF와 절세 관련 내용은 안내와 고지 중심으로 제공한다.

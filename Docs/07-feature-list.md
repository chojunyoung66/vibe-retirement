# 07 Feature List

## Guide

아래 목록은 `Docs/` 문서 전체에서 나온 기능·아이디어를 Agent가 다시 추출한 것입니다. 우선순위, MVP 여부, 제외 여부를 판단하지 않고, 문서에 등장한 항목을 기능 후보로 풀어 F-01부터 일렬로 다시 번호를 붙였습니다.

규칙:

- docs 어디에든 나온 기능·아이디어를 빠짐없이 포함합니다.
- 목적·가치, 여정, Pain/Wow, 가설, 키워드, 질문·답변에서 나온 아이디어도 포함합니다.
- F-01부터 번호를 매겨 일렬로 나열합니다.
- 번호와 순서는 우선순위가 아닙니다.
- 기능명은 짧고 구체적으로 한 줄로 작성합니다.
- 우선순위 판단, 제외, MVP 선별을 하지 않습니다.
- 의심스러워도 일단 나열합니다.

## 4. 현재 권장 MVP 범위

### MVP 1차

- F-01 개인별 은퇴 목표 및 현재 재정 상태 입력
- F-02 건강보험료 예상 시뮬레이션
- F-09 은퇴 후 월별 현금흐름 시뮬레이션
- F-10 은퇴 자산 고갈 시점 예측
- F-11 은퇴 계획 진행 상황 대시보드

### 2차 확장

- F-03 ISA 계좌 개설 및 운용 가이드

- F-05 퇴직연금 현황 분석
- F-06 개인연금 포트폴리오 진단
- F-07 ETF 포트폴리오 구성
- F-08 투자 성향 분석

### 3차 고도화

- F-12 전문가 상담 연결
- F-13 최신 세법 및 금융 상품 정보 제공
- F-14 은퇴 준비 체크리스트 및 목표 달성 알림

## 5. 개발 우선순위

| 우선순위 | 대상 | 이유 |
|---|---|---|
| P0 | 공통 데이터 모델 | 계산 기능 전체의 기반 |
| P0 | 공통 고지문 | 세금·건보료·투자 결과의 책임 범위 명확화 |
| P0 | F-01 | 모든 기능의 입력 기반 |
| P0 | F-09 | 서비스 핵심 가치인 월별 현금흐름 확인 |
| P1 | F-02 | 은퇴 후 주요 고정지출 반영 |
| P1 | F-10 | 자산 지속가능성 판단 |
| P1 | F-11 | 결과를 한눈에 보여주는 대시보드 |
| P2 | F-03~F-08 | 절세·연금·투자 확장 기능 |
| P3 | F-12~F-14 | 운영·상담·알림 고도화 기능 |

## 6. 공통 주의사항

- 세금, 건강보험료, 투자 결과는 모두 추정치입니다.
- 결과 화면에는 기준일, 계산 가정, 고지문을 표시해야 합니다.
- 초기 MVP에서는 자동 연동보다 사용자가 직접 입력하고 결과를 확인하는 흐름을 우선합니다.
- 투자 관련 기능은 개별 상품 매수·매도 권유가 아니라 정보 제공 및 예시 중심으로 제한합니다.
- 최신 정보 제공 기능은 공식 출처, 기준일, 업데이트일을 반드시 표시해야 합니다.

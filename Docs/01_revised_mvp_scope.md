# 은퇴현금 서비스 MVP 범위 재정의

## 1. MVP 재정의 목적

기존 README와 F-01~F-14 문서는 종합 은퇴 설계 플랫폼을 목표로 하고 있습니다. 그러나 초기 개발 단계에서는 사용자가 가장 빠르게 가치를 느낄 수 있는 핵심 문제에 집중해야 합니다.

핵심 질문:

> 은퇴 후 매달 생활비가 충분한가?
> 언제 현금이 부족해질 수 있는가?
> 지금 무엇을 조정해야 하는가?

## 2. MVP 1차 범위

### F-01 개인별 은퇴 목표 및 현재 재정 상태 입력

역할:

- 은퇴 시점, 월 생활비, 현재 자산, 연금, 부채, 지출 정보를 입력받는 기본 화면
- 모든 계산 기능의 출발점

MVP 포함 항목:

- 출생연도 또는 현재 나이
- 퇴직 예정 연도/월
- 은퇴 후 월 생활비
- 국민연금 예상 수령액 및 시작 시점
- 퇴직연금/개인연금/금융자산/부동산 자산
- 월 고정지출 및 변동지출

MVP 제외 항목:

- 복잡한 상품별 세부 진단
- 금융기관 연동
- 자동 자산 조회

### F-02 건강보험료 예상 시뮬레이션

역할:

- 은퇴 후 고정지출 중 큰 비중을 차지하는 건강보험료를 추정

MVP 포함 항목:

- 지역가입자 예상 보험료 단순 추정
- 임의계속가입 안내
- 피부양자 가능성 안내
- 장기요양보험료 포함 여부 표시

MVP 제외 항목:

- 건강보험공단 실시간 산식 연동
- 개인별 정확 보험료 산정

### F-09 은퇴 후 월별 현금흐름 시뮬레이션

역할:

- 월별 수입, 지출, 순현금흐름을 계산하는 핵심 기능

MVP 포함 항목:

- 연금 수입
- 금융자산 인출
- 생활비/의료비/건강보험료/재산세 등 지출
- 월별 순현금흐름
- 누적 잔액

MVP 제외 항목:

- 복잡한 세후 계산
- 금융상품별 배당/이자 자동 계산
- 외부 API 연동

### F-10 은퇴 자산 고갈 시점 예측

역할:

- 현재 자산과 월별 현금흐름 기준으로 자산이 언제 부족해지는지 예측

MVP 포함 항목:

- 고갈 예상 연도/나이
- 기본 시나리오
- 생활비 증가 시나리오
- 수익률 하락 시나리오

MVP 제외 항목:

- 확률 기반 몬테카를로 분석
- 정교한 투자 수익률 모델

### F-11 은퇴 계획 진행 상황 대시보드

역할:

- 사용자가 입력한 정보와 계산 결과를 한 화면에서 확인

MVP 포함 항목:

- 월 수입/지출/순현금흐름 요약
- 예상 자산 고갈 시점
- 건강보험료 추정치
- 은퇴 준비 상태 메시지

MVP 제외 항목:

- 장기 리포트 자동 생성
- 알림/뉴스레터
- 계좌 연동 대시보드

## 3. 2차 기능 범위

### 절세/연금 정보 확장

- F-03 ISA 계좌 개설 및 운용 가이드
- F-04 ISA 절세 효과 분석
- F-05 퇴직연금 현황 분석
- F-06 개인연금 포트폴리오 진단

2차에서는 실제 상품 추천보다 정보 제공과 간단한 시뮬레이션 수준으로 시작합니다.

### 투자 기능 확장

- F-08 투자 성향 분석
- F-07 ETF 포트폴리오 구성

역할 분리:

- F-08: 성향 진단
- F-07: 성향 결과를 기반으로 한 자산군 배분 예시

## 4. 3차 고도화 기능 범위

- F-12 전문가 상담 연결
- F-13 최신 세법 및 금융 상품 정보 제공
- F-14 은퇴 준비 체크리스트 및 목표 달성 알림

이 기능들은 운영 체계, 콘텐츠 관리, 알림 동의, 개인정보 처리 등 추가 준비가 필요하므로 MVP 이후로 분리합니다.

## 5. MVP 성공 기준

사용자가 다음 질문에 답을 얻을 수 있으면 MVP 성공으로 봅니다.

1. 내가 은퇴 후 매달 얼마가 부족하거나 남는가?
2. 내 자산은 몇 세까지 버틸 수 있는가?
3. 건강보험료 같은 고정지출을 감안했는가?
4. 생활비를 줄이면 자산 고갈 시점이 얼마나 늦춰지는가?
5. 다음에 무엇을 입력하거나 조정해야 하는가?

## 6. 개발 우선순위

1. 공통 데이터 모델 정의
2. F-01 입력 화면
3. F-09 월별 현금흐름 계산
4. F-10 자산 고갈 시점 계산
5. F-02 건강보험료 추정
6. F-11 대시보드
7. 고지문/기준일/계산 가정 표시
